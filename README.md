# Frame Size Recommendation Tool

This is a web-based tool to help users find their perfect eyeglass frame size based on their face measurements.

## Files
- `index.html`: The main structured page.
- `style.css`: Styling and layout.
- `script.js`: Logic for calculations and interactivity.

## How to Use
1. **Unzip** the folder if you received this as a zip file.
2. Locate the `index.html` file in the folder.
3. **Double-click** `index.html` to open it in your default web browser (Chrome, Safari, Edge, etc.).
4. The application will launch immediately. No internet connection or server installation is required.
